== Description ==

A stylish WordPress countdown/countup timer plugin

**Requirements:** WordPress 4.0+, PHP5.3 and higher

Whether you want to stir up some excitement for a event or mark down an important milestone, WP Time Counter has your back with customizable count-downs/count-ups. Just create a counter & a shortcode will be automatically generated for you which you can use in page/post. You can also to show up a custom message after the countdown is over. Comes with 6 beautiful skins.

This plugin is free to use and is not actively supported by the author, but will be monitored for serious bugs that may need corrections.

== Installation ==

1. Upload the 'wp-time-counter' folder and all its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a page or post - you will find Time Counter button. Click on it and choose the settings and click on Add Counter. You are done!
